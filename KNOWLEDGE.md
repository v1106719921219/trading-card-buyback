# 買取システム 操作マニュアル (KNOWLEDGE.md)

## 申し込みフロー

### お客様の申し込み手順
1. `/apply` にアクセスして買取希望商品を選択・数量入力
2. 発送先の事務所を選択
3. お客様情報（名前・連絡先）を入力して送信
4. 申し込み完了後、**注文番号**が発行される

### 申し込み完了後の編集方法
- **編集可能な期間**: ステータスが「申込」の間のみ編集可能
- **編集ページ**: `/apply/complete?order_number=注文番号&office_id=事務所ID`
  - 商品の追加・削除・数量変更ができる
  - 「保存」ボタンで変更を反映
- **追跡番号の入力**: 同じ完了ページで発送後に追跡番号を入力できる

## 追跡・状態確認

### お客様が注文を追跡する方法
- `/tracking` にアクセスして注文番号を入力
- 注文状態・発送状況が確認できる

## 管理者向け操作

### 注文管理 (`/admin`)
- 申し込み一覧の確認・ステータス変更
- 支払い確認 (`/admin/payment-verification`)
- 入荷予定管理 (`/admin/arrival-schedule`)

### 商品管理
- 買取価格の設定・更新 (`/admin/products`)
- 一括価格更新 (`/admin/products/bulk-update`)

## よくある質問

**Q: 申し込み後に商品を追加したい**
→ 申し込み完了メールの完了ページURL（`/apply/complete?order_number=...`）にアクセスして商品を追加・保存

**Q: ステータスが「申込」以外で編集できない**
→ 処理が進んでいるため編集不可。管理者に相談してください

**Q: 追跡番号はどこで入力する？**
→ 申し込み完了ページ（`/apply/complete?order_number=...`）の下部に追跡番号入力欄があります
